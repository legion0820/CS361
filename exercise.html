This is the Exercise Page

This is a page to choose which exercise you would like to do. You can combine different exercise from different category or if you would like, you can do a push, pull, legs days.

Push day is any exercise that requires using the motion to push away from your body like chest exercises, triceps, and shoulders.

Pull day is any exercise that requires using the motion to pull towards your body like back exercises and biceps.

Leg day is any exercise that requires using the motion of your legs like leg exercises.
